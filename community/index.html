Apache Spot Mailing Lists and Chat Rooms

Get help using Spot or contribute to the project on our mailing lists or our chat room:

Get Help
  • JIRA Issue Tracker
  • user@spot.incubator.apache.org (subscribe) (unsubscribe) (archives) - for usage questions, help, and announcements.
  • issues@spot.incubator.apache.org (subscribe) (unsubscribe) (archives) - receives an email notification for all ticket updates made in the Spot JIRA issuetracker.
Developer Mailing Lists
  • dev@spot.incubator.apache.org (subscribe) (unsubscribe) (archives) - for people who want to contribute code to Spot.
  • commits@spot.incubator.apache.org (subscribe) (unsubscribe) (archives) - receives an email notification of all code changes to the Spot Git repository.
Community Resources
  • GitHub - where users and committers will find tools and technical information to commit.
  • Confluence Site - where deevelopers will find project details such as roadmap, meeting notes, and other useful write-ups

More Info

Apache Spot is an effort undergoing incubation at The Apache Software Foundation (ASF), sponsored by the Apache Incubator. Incubation is required of all newly accepted projects until a further review indicates that the infrastructure, communications, and decision making process have stabilized in a manner consistent with other successful ASF projects. While incubation status is not necessarily a reflection of the completeness or stability of the code, it does indicate that the project has yet to be fully endorsed by the ASF.
